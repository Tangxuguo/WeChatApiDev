WeChatApiDev
============

Django Sae Weixin
本程序使用了Django框架，部署在SAE上面，针对微信公共平台进行开发实现了以下功能：
Menu自定义菜单:
“豆瓣”：返回图文信息
1：豆瓣电影，信息查询
2：豆瓣音乐，信息查询
3：豆瓣图书，信息查询
“扇贝”：
4：扇贝新闻
5：扇贝读书
6：扇贝小组
7：扇贝论坛
“帮助”
“翻译”：直接输入文本
